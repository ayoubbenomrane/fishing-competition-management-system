# 🎣 Fishing Competition Score App

A full-stack web application designed to manage multi-day fishing tournaments.  
Built from scratch — frontend and backend — to track players, their catches, and dynamic rankings over time.

---

## 🧑‍💻 Solo Full-Stack Project

> I was fully responsible for the **entire development** of this app:  
> - Frontend: built with HTML, CSS, and JavaScript (no framework)  
> - Backend: powered by Node.js, Express, and PostgreSQL

---

## 🧩 Features

- 🐟 **Player Management**
  - Add, edit, and delete players
  - Modal-based UI and in-table editing

- 📆 **Multi-Day Journée Tracking**
  - Record number of fish, total weight per player, per day
  - Automatically assign scores per journée

- 🧮 **Ranking System**
  - Calculate and display live rankings per day
  - Compute **final overall ranking** across all days
  - Handle absences with penalty scores

- 📑 **Persistent Storage**
  - PostgreSQL schema with raw SQL queries
  - Pre-calculated scores and ranking logic handled server-side

---

## 🛠 Tech Stack

- **Frontend**: HTML, CSS, Vanilla JavaScript  
- **Backend**: Node.js, Express.js  
- **Database**: PostgreSQL  
- **Other Tools**: pgAdmin, Postman (for testing), raw SQL (no ORM)
